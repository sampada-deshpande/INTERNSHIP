StockStream – Retail Inventory & Billing System

 Project Overview
StockStream is a Python-based Point of Sale (POS) system developed using
Tkinter. The application helps manage retail inventory, handle customer
billing, apply GST, and generate receipts automatically.

The project is designed with clean logic separation so that inventory,
billing, UI, and receipt generation are handled in different modules.
This makes the code easy to understand, maintain, and explain.

Problem Statement

Retail stores require a simple yet reliable system to:
Track available inventory
Prevent selling products with low stock
Calculate bills accurately with tax
Generate receipts for every transaction
StockStream solves these problems by providing an easy-to-use graphical interface combined with clean backend logic.
